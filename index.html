Pick a Picture

Technologies used: Python, Django, SQLite3, jQuery, HTML, CSS

This full stack Python application is a logic game for kids. Each time the button is clicked, four pictures are randomly selected from the database in a way that one of them doesn't belong to the set, based on some criteria. The criteria is also selected randomly each time. The goal of the game is to pick the correct picture.

This project won a Coding Dojo hackathon in nomination "Implementation of Technology".
